# USAGE

## autoepbm

### INSTALATION

Recommended OS : linux 
1. install selenium 
`$ pip install selenium` 
2. install firefox webdriver 
`https://github.com/mozilla/geckodriver/releases` 
3. copy the "geckodriver" file to /usr/bin/ 
4. run the script
`$ python autoepbm.py`
5. enter your login from the terminal
